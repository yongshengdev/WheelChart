# WheelChart
**Usage**

`compile 'com.sign.wheelchart:wheelchart:1.0.0'`

**Features** 
- 绘制横向图表  
- 可拖动并处理惯性滚动事件  
- 拖动或惯性滚动结束后，自动选中离中心最近的下标   
- 点击图表，根据点击区域选中离点击区域最近的下标  
- 处理嵌套滚动
